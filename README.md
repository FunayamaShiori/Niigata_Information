# 動作確認方法
## カートに入れる
1. Topページ右上のログインからテスト用管理者アカウントでログイン
  ※ログインしないとカートページに遷移しません
2. 商品の数量を選択して、『カートに入れる』ボタンを押す
3. 確認画面に遷移するので、購入完了ボタンを押すと、Cartsテーブルに保存され、管理者ページにも反映されます。

## 管理者ページ
1. 注文リストの右側『顧客情報』を押すと顧客情報が出てくる
2. 管理者のみedit(編集)できます
3. ユーザー一覧も確認できます

# cartsテーブル
